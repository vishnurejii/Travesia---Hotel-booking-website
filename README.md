# Travesia-Hotel-Booking-Website
🏨 Travesia – Hotel Booking Management System
📌 Overview

Travesia is a hotel booking web application where users can browse hotels and make bookings. It also supports a role-based system where hotel owners can add and manage their own hotel listings.

The project focuses mainly on backend development, including API design, authentication, and database handling.

🚀 Features
👤 User Side
User signup and login
Browse hotels
View hotel details
Book hotels
🏨 Hotel Owner Side
Add new hotels
Update hotel details
Delete hotel listings
Manage their own hotels
🛠️ Tech Stack
Node.js
Express.js
MongoDB
React
REST APIs
JWT Authentication
🔐 Key Highlights
Implemented secure authentication using JWT
Built REST APIs for all core functionalities
Designed role-based access (user vs hotel owner)
Handled database operations using MongoDB
Worked on error handling and debugging

Deployed and accessible online.
